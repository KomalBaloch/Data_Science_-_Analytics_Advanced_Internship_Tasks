# 📊 Term Deposit Subscription Prediction

## 🎯 Objective
Predict whether a bank customer will subscribe to a term deposit based on marketing campaign data.

---

## 📁 Dataset
Bank Marketing Dataset (UCI Machine Learning Repository)

---

## 🧠 Problem Type
Binary Classification (Yes / No)

---

## ⚙️ Steps Performed
- Data Loading & Exploration
- Data Cleaning
- Feature Encoding (One-Hot Encoding)
- Model Training (Logistic Regression, Random Forest)
- Model Evaluation (Confusion Matrix, F1 Score, ROC Curve)
- Explainable AI using SHAP

---

## 📊 Results
- Best Model: Random Forest
- Evaluation Metrics: F1-score, ROC-AUC
- SHAP used for interpretability

---

## 🔍 Key Insights
- Duration of last contact is most important feature
- Balance affects subscription probability
- Housing loan reduces subscription chance

---

## 📌 Tools Used
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn, SHAP
